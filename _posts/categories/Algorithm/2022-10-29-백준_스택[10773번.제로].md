0을 입력받는순간, pop되는 것을 이용한다.

## 🍳풀이법
```python
def stack_zero():
    k = int(input())
    stack = []
    sum_ = 0
    for i in range(k):
        input_ = int(input())
        if input_ == 0 and stack:
            stack.pop()
        else:
            stack.append(input_)

    for stk in stack:
        if not stack:
            return print(0)
        sum_ += stk

    return print(sum_)

stack_zero()
```
이번에도 금방 풀었으나, 시간복잡도를 고려해봐야 겠다는 생각도 들었다.
4000ms 정도가 나오는게 너무나 이상해서 한번 확인해 볼 필요는 있겠다.
