## 🐋풀이법
```python
def queue_algorithm():
    num = int(input())
    queue = []
    for i in range(num):
        queue.append(num-i)
    print(queue)

    while True:
        if len(queue) >= 2:
            queue.pop()
            print(queue)
        if len(queue) == 1:
            break
        temp = queue[-1]
        queue.pop()
        queue.insert(0, temp)

        print(queue)
    return print(queue[0])


queue_algorithm()
```

## 🤔고민 많이했던 부분
```python
temp = queue[-1]
queue.pop()
queue.insert(0, temp)
```

먼저, 로직을 짰던 것을 설명해보면 queue 배열에 들어있는 마지막 카드를 temp 변수에 저장해둔다.

그리고나서 옮긴 변수 자리는 pop을 통해 빼내고, temp에 저장된 변수를 insert 메서드를 이용하여 집어넣는다. 앞에 위치해야 하므로

```python
queue.insert(0,temp)
```
이렇게 넣어주었다.
